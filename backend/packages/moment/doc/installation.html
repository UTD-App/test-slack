UTD Package · utd/moment

Installing the Moment package

Manual drop-in — Backend (Laravel) + Flutter

Installation (English)

Moment is a standalone composer package (utd/moment, namespace Utd\Moment) installed manually as a drop-in — it is not an nwidart module and is not auto-discovered. Requires PHP 8.2+. Install both halves on top of an existing UTD base.

Backend (Laravel)

  1. 1. Copy the package folder Place the backend package at backend/packages/moment/.
  2. 2. Register the PSR-4 autoload (root composer.json) Add under autoload.psr-4 in backend/composer.json (and the test namespace under autoload-dev.psr-4 if you run the package tests): 
    "autoload": {
  "psr-4": {
    "Utd\\Moment\\": "packages/moment/src/"
  }
},
"autoload-dev": {
  "psr-4": {
    "Utd\\Moment\\Tests\\": "packages/moment/tests/"
  }
}
  3. 3. Register the service provider Add to the providers list (in config/app.php or bootstrap/providers.php): 
    Utd\Moment\Providers\MomentServiceProvider::class,
    The provider auto-wires config, migrations, views, translations and the API routes, and self-registers the package in the base PackageRegistry. (The provider is also listed under extra.laravel.providers in the package's composer.json, so it is auto-discovered when the package is required via Composer.)
  4. 4. Register the Filament admin plugin In app/Providers/Filament/AdminPanelProvider.php add to the panel: 
    ->plugin(\Utd\Moment\Filament\MomentPlugin::make())
  5. 5. Dump autoload & run migrations 
    composer dump-autoload
php artisan migrate
    Migrations create moment, moment_galleries, moment_user_likes, moment_user_comments, moment_comment_likes, report_moments and report_moment_comments.
  6. 6. Sync the package (required) 
    php artisan utd:sync-packages
    Writes the package row into the packages table so it appears in admin/packages (enable / disable) and in /api/packages/installed (which the app reads). Run after every install/update.
  7. 7. Enable in admin/packages Open admin/packages and toggle Moment on. While it is disabled, the API routes return 403 (the package.enabled:moment middleware) and the admin page hides itself.
Verify: 
php artisan route:list --path=moment
php vendor/bin/phpunit packages/moment/tests --no-coverage

Flutter

  1. 1. Copy the package & fix its path dep Place the Flutter package at flutter/packages/moment/ (singular). In packages/moment/pubspec.yaml, point the utd_app path at your base app root.
  2. 2. Add it as a path dependency In the host app's flutter/pubspec.yaml: 
    dependencies:
  moment:
    path: packages/moment
  3. 3. Register MomentFeature() In lib/main.dart → buildFeatures(): 
    import 'package:moment/moment.dart';

List<AppFeature> buildFeatures() => [
  AuthFeature(),
  MomentFeature(),   // ← the moments feed
];
    MomentFeature registers the feed routes, Stac SDUI parsers/data sources, and merges the moment.* UI translations (admin/server overrides win).
  4. 4. Fetch packages & run 
    cd flutter
flutter pub get
flutter run
Naming: the Composer name is utd/moment and the Flutter package is moment (singular). Import is package:moment/moment.dart.

Optional: gifting

Gifting on moments auto-enables once the Gifts package is installed and binds App\Contracts\GiftDirectory — no extra step in Moment. Without it, gift routes return a safe 503 and the rest of the feed works.

Uninstall

Reverse steps 2/3/4, delete the folder, run composer dump-autoload, and remove the package row from admin/packages. Roll back the migrations first if you want the tables gone.

التثبيت (بالعربية)

Moment حزمة Composer مستقلة (utd/moment, namespace Utd\Moment) بتتسطّب يدوي (drop-in) — مش nwidart module ومش بتتكتشف أوتوماتيك. بتحتاج PHP 8.2+. ركّب النُصّين فوق قاعدة UTD مثبّتة.

الـ Backend (Laravel)

  1. 1. انسخ فولدر الحزمة حُط حزمة الـ backend في backend/packages/moment/.
  2. 2. سجّل الـ PSR-4 autoload (في الـ root composer.json) ضيف تحت autoload.psr-4 في backend/composer.json (و namespace الاختبارات تحت autoload-dev.psr-4 لو هتشغّل اختبارات الحزمة): 
    "autoload": {
  "psr-4": {
    "Utd\\Moment\\": "packages/moment/src/"
  }
},
"autoload-dev": {
  "psr-4": {
    "Utd\\Moment\\Tests\\": "packages/moment/tests/"
  }
}
  3. 3. سجّل الـ Service Provider ضيف في قائمة providers (في config/app.php أو bootstrap/providers.php): 
    Utd\Moment\Providers\MomentServiceProvider::class,
    الـ provider بيظبط الـ config والـ migrations والـ views والترجمات ومسارات الـ API أوتوماتيك, وبيسجّل الحزمة في PackageRegistry. (الـ provider مكتوب كمان تحت extra.laravel.providers في composer.json بتاع الحزمة, فبيتكتشف أوتوماتيك لو الحزمة اتعملها composer require.)
  4. 4. سجّل بلَجن الأدمن (Filament) في app/Providers/Filament/AdminPanelProvider.php ضيف للـ panel: 
    ->plugin(\Utd\Moment\Filament\MomentPlugin::make())
  5. 5. حدّث الـ autoload وشغّل الـ migrations 
    composer dump-autoload
php artisan migrate
    الـ migrations بتنشئ moment, moment_galleries, moment_user_likes, moment_user_comments, moment_comment_likes, report_moments, report_moment_comments.
  6. 6. سجّل الحزمة في النظام (مطلوب) 
    php artisan utd:sync-packages
    بيكتب صف الحزمة في جدول packages عشان تبان في admin/packages (تشغيل/إطفاء) وفي /api/packages/installed (اللي التطبيق بيقرا منه). شغّلها بعد كل تثبيت أو تحديث.
  7. 7. فعّلها من admin/packages افتح admin/packages وشغّل Moment. وهي معطّلة, مسارات الـ API بترجع 403 (الميدلوير package.enabled:moment) وصفحة الأدمن بتختفي.
التحقق: 
php artisan route:list --path=moment
php vendor/bin/phpunit packages/moment/tests --no-coverage

الـ Flutter

  1. 1. انسخ الحزمة وظبّط مسار الاعتماد حُط حزمة الـ Flutter في flutter/packages/moment/ (بالمفرد). في packages/moment/pubspec.yaml خلّي مسار utd_app يشاور على جذر تطبيق القاعدة.
  2. 2. ضيفها كـ path dependency في flutter/pubspec.yaml بتاع التطبيق: 
    dependencies:
  moment:
    path: packages/moment
  3. 3. سجّل MomentFeature() في lib/main.dart ← buildFeatures(): 
    import 'package:moment/moment.dart';

List<AppFeature> buildFeatures() => [
  AuthFeature(),
  MomentFeature(),   // ← فيد اللحظات
];
    MomentFeature بيسجّل مسارات الفيد, ومعالِجات/مصادر بيانات Stac (SDUI), وبيدمج ترجمات الـ moment.* (الأدمن/السيرفر بيكسبوا في الأولوية).
  4. 4. حمّل الحزم وشغّل التطبيق 
    cd flutter
flutter pub get
flutter run
تنبيه على الاسم: اسم Composer هو utd/moment وحزمة الـ Flutter اسمها moment (بالمفرد). الاستيراد package:moment/moment.dart.

اختياري: الهدايا

الهدايا على اللحظات بتتفعّل أوتوماتيك بمجرد تثبيت حزمة Gifts وربطها App\Contracts\GiftDirectory — مفيش خطوة زيادة في Moment. من غيرها مسارات الهدايا بترجع 503 بأمان وباقي الفيد شغّال.

الإلغاء (Uninstall)

اعكس الخطوات 2/3/4, امسح الفولدر, شغّل composer dump-autoload, وامسح صف الحزمة من admin/packages. لو عايز تشيل الجداول اعمل rollback للـ migrations الأول.
